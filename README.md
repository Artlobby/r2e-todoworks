# r2e-todoworks

todoworks for desktop use React, Electron and Mobx

## Development
```
$ npm install
$ npm run dev
```

Generate the binary:
* For Linux
```
$ npm run package-linux
```
Maybe you will install some depends packages.
* For Mac
```
$ npm run package-mac
```
After that, you will see the binary in `./release` folder
